# wow-flex-box

一组简单实用的语义化 flex 样式类，通过它们基本可以组合出所有常见的业务布局

## 安装

```
npm install wow-flex-box
```

## 使用

### JavaScript 中引入

```javascript
import "wow-flex-box";
```

### Less 中引入

```less
@import "wow-flex-box";
```
